## 🚀 Quick Start Guide

### Prerequisites

- **Docker** and **Docker Compose** installed on your machine
  - Windows/Mac: [Docker Desktop](https://www.docker.com/products/docker-desktop/)
  - Linux: [Docker Engine](https://docs.docker.com/engine/install/)
- A Paradigm API key (get it from [paradigm.lighton.ai](https://paradigm.lighton.ai))

### Installation Steps

**1. Configure your API key**

Create a `.env` file at the project root:

```bash
# On Windows (PowerShell)
copy .env.example .env

# On Mac/Linux
cp .env.example .env
```

Edit the `.env` file and replace `your_paradigm_api_key_here` with your actual API key:

```env
PARADIGM_API_KEY=sk-your-api-key-here
```

**2. Start the application**

```bash
docker-compose up -d
```

**This command will:**
- Build the Docker image
- Install all dependencies
- Start the application in the background

**3. Access the interface**

Open your browser and go to:

```
http://localhost:8000
```

You will see your workflow interface!

## 🔧 Troubleshooting

### Issue: "Port 8000 already in use"

Another service is already using port 8000. Solutions:

1. **Stop the other service** or
2. **Change the port** in `docker-compose.yml`:
   ```yaml
   ports:
     - "8001:8000"  # Use port 8001 instead
   ```
   Then access `http://localhost:8001`

### Issue: "API key not configured"

Check that:
1. The `.env` file exists at the project root
2. The API key is correct (starts with `sk-...`)
3. You restarted the application after modifying `.env`

```bash
docker-compose down
docker-compose up -d
```

### Issue: "Failed to upload file"

Check that:
1. Your Paradigm API key is valid
2. You have internet access
3. The file is not too large (limit: 100 MB)
